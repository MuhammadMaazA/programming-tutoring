# Year 1 Programming Tutoring

Welcome! This repository contains all the materials for our weekly programming tutoring sessions.

## Repository Structure

- **weekly-sessions/** - Notes and code from each session
- **questions/** - Practice problems and exercises
- **resources/** - Helpful guides and reference materials
- **solutions/** - Solutions to practice questions

## Schedule

We meet weekly throughout Term 1 and Term 2. Check the WhatsApp group for our meeting time and any updates.

## Getting Started

1. Star this repo to stay updated
2. Check the relevant folders before each session
3. Try the practice questions on your own
4. Bring questions to our sessions!

## Contact

Got questions? Message me on the WhatsApp group or reach out directly.
